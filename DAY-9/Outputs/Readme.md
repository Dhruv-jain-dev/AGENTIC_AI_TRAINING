# Day 9 – Constructors and Circular Linked Lists in Python

**Date:** 4 July 2026

## Overview

The ninth day of the Agentic AI Training focused on advanced Object-Oriented Programming concepts and data structures in Python. We explored different ways of creating constructors and understood how constructors can be customized to initialize objects in various scenarios. The session then introduced the implementation of a **Circular Linked List**, demonstrated through a song playlist application. To strengthen my understanding, I independently implemented a Flight Management System using a Circular Linked List and performed various insertion and deletion operations.

* Revised the concept of constructors and their role in object initialization.
* Learned different ways of defining constructors in Python.

---

## 2. Circular Linked List

* Learned the concept of a **Circular Linked List**, where the last node points back to the first node instead of containing a `NULL` reference.
* Understood how circular linking enables continuous traversal through the list.
* Studied the structure of nodes containing data and a reference to the next node.
* Discussed the advantages and practical applications of Circular Linked Lists in real-world systems.

---

## 3. Song Playlist Implementation

* Implemented a **Song Playlist** using a Circular Linked List.
* Learned how each song is represented as a node connected in a circular manner.
* Understood how continuous playback can be achieved by traversing the circular list repeatedly.
* Explored how insertion and deletion operations are performed while maintaining the circular structure.
