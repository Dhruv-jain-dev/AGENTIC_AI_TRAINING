# Day 13 – MongoDB Setup, Database Connection, and Data Insertion with Python

**Date:** 10 July 2026

---

## Overview

The thirteenth day of the Agentic AI Training introduced us to **MongoDB**, one of the most popular NoSQL databases used in modern application development. We learned the fundamentals of MongoDB, its architecture, and how it differs from traditional relational databases. The session also covered the complete setup process, including creating a MongoDB Atlas cluster, configuring the database, establishing a connection between Python and MongoDB, and inserting data into the database through Python code.
