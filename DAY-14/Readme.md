# Day 14 – MongoDB CRUD Operations and Introduction to Streamlit

**Date:** 13 July 2026

---

## Overview

The fourteenth day of the Agentic AI Training focused on performing **CRUD (Create, Read, Update, Delete)** operations in MongoDB using Python and getting started with **Streamlit**, a Python framework for building interactive web applications. We learned how to update and delete documents stored in a MongoDB database through Python code. The session also introduced the fundamentals of Streamlit, its role in creating user interfaces, and its advantages for rapidly developing web applications. To reinforce these concepts, we built a simple **User Registration Form** using Streamlit.
