# Day 2 – Python Fundamentals & Memory Management

**Date:** 26 June 2026

## Overview

The second day of the Agentic AI Training focused on building a strong foundation in Python programming and understanding how Python programs execute internally. We also explored memory management concepts, threading, and revisited the MVC (Model-View-Controller) architecture with a deeper understanding of data organization.

---

## Topics Covered

### 1. Python Basics

* Introduced to the fundamental concepts of Python.
* Discussed Python syntax and its beginner-friendly nature.
* Learned why Python is widely used in AI, Machine Learning, Data Science, and Automation.

### 2. Python Program Execution

* Understood the lifecycle of a Python program.
* Learned how the Python interpreter reads, compiles, and executes Python code.
* Discussed the step-by-step processing involved when a Python program is run.

### 3. Variable Storage in RAM

* Learned how variables are stored in the computer's memory (RAM).
* Understood that every variable occupies a memory location.
* Discussed how Python manages memory efficiently during program execution.

### 4. Stack and Heap Memory

* Learned that RAM is logically divided into:

  * **Stack Memory:** Stores function calls, local variables, and execution information.
  * **Heap Memory:** Stores dynamically allocated objects and data.
* Understood the purpose of both memory regions and how Python utilizes them.

### 5. Threads in Python

* Introduced to the concept of **Main Thread**.
* Learned about **Child Threads** and how multiple threads can perform tasks concurrently.
* Discussed the importance of threading in improving application performance.

### 6. MVC (Model-View-Controller) Architecture

Revisited the MVC architecture with a focus on the **Model** component.

* **Model:** Responsible for managing application data.
* Data within the Model can be classified into:

  * **Single-Valued Container**
  * **Multi-Valued Container**

    * **Homogeneous Containers:** Store elements of the same data type.
    * **Heterogeneous Containers:** Store elements of different data types.

This classification helps in organizing and managing data efficiently within an application.

## Key Takeaways

* Built a strong foundation in Python programming.

* Learned how variables are stored and managed in RAM.
* Gained knowledge of Stack and Heap memory.
* Explored the concepts of Main Thread and Child Threads.
* Deepened understanding of the Model component in MVC architecture and different data container types.
